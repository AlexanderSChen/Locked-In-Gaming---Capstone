# Locked-In-Gaming---Capstone
## This site use Free-To-Play Games external API
**Live Site URL: https://rapidapi.com/digiwalls/api/free-to-play-games-database/**
### The data is feteched from this API, the information is displayed on the game details page and on the cards.
---
Users can create an account, leave a comment on the game's details page, favorite each other's statuses, follow each other's profiles and view each other's statuses.
Game statuses will be displayed under the respective games and on the user's profiles.
